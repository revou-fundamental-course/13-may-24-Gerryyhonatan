LUAS SEGITIGA

RUMUS LUAS SEGITIGA YAITU

L = 1/2 x a x t

L = luas

a = panjang alas

t = tinggi

HASIL LUAS SEGITIGA

KELILING SEGITIGA

RUMUS KELILING SEGITIGA YAITU

K = S1 + S2 + S3

K = keliling

S1 = sisi A

S2 = sisi B

S3 = sisi C

HASIL KELILING SEGITIGA

LUAS JAJAR GENJANG

RUMUS LUAS JAJAR GENJANG YAITU

L = a x t

L = luas

a = panjang alas

t = tinggi

HASIL LUAS JAJAR GENJANG

KELILING JAJAR GENJANG

RUMUS KELILING JAJAR GENJANG YAITU

K = 2a + 2b

K = keliling

a = sisi a

b = sisi b

HASIL KELILING JAJAR GENJANG
